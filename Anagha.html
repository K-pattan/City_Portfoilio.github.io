Algorithm Codes (Reference)

Breadth First Search (BFS)

Create an empty queue
Mark start node as visited
Insert start node into queue

while queue is not empty:
  remove front node
  visit all unvisited adjacent nodes
  mark them visited
  insert them into queue

Depth First Search (DFS)

Mark start node as visited
for each adjacent node:
  if not visited:
    perform DFS on that node

Dijkstra’s Algorithm

Initialize distance of all nodes as infinity
Set distance of source node = 0

repeat:
  choose unvisited node with smallest distance
  mark it visited
  update distances of its neighbors
until all nodes are visited

Prim / Kruskal (Minimum Spanning Tree)

Sort all edges by increasing weight
Select smallest edge
Add edge if it does not form a cycle
Repeat until all nodes are connected

Greedy Algorithm

Select the locally optimal choice
Add it to solution
Repeat until problem is solved

Segment Tree

Build tree from input array
Each node stores range information

For query:
  Traverse required segments

For update:
  Update value and propagate changes

Priority Queue

Insert elements with priority
Remove element with highest priority first
Repeat until queue is empty

Queue

Insert element at rear (enqueue)
Remove element from front (dequeue)

Hashing

Apply hash function to key
Store value at computed index
Retrieve value using same hash function

Trie

Insert characters of word one by one
Each character represents a node
Search word by traversing characters

Heap

Insert element into heap
Rearrange to maintain heap property
Delete root element when required

Sorting (General)

Divide array into smaller parts
Sort each part
Merge sorted parts

Union-Find

Find parent of each node
Union two sets if parents differ
Used to check connectivity
