Telecommunication Services

Telecommunication services involve the transmission of information over long distances using technologies such as phones, internet networks, satellites, and wireless systems.

1. High speed Fiber & 5G network: This idea aligns with the United Nations Sustainable Development Goal (SDG) 9 – Industry, Innovation & Infrastructure.
It also indirectly supports:
• SDG 8 – Decent Work & Economic Growth
• SDG 11 – Sustainable Cities & Communities
• SDG 4 – Quality Education (through digital learning access)

How It Solves Citizens and Local Business Problems:

  • Provide reliable high-speed connectivity enabling online education, remote work & IT jobs
  • Fast connection enables e-commerce, digital payments, online marketing
  • Network infrastructure attracts startups & investors
  • 5G supports real-time response, IoT safety systems, surveillance
  • New opportunities encourage jobs & entrepreneurship locally
  • Enables smart governance, e-health, online documentation



Algorithm/data structure used:

  • Arrays:
    An array is a linear data structure that stores elements of the same data type at contiguous memory locations.
    Stores network nodes, IP allocations, bandwidth usage records.
    Space Efficiency:Low
    Time Efficiency: 0(1) fast access

  • Linked list:
    A linked list is a data structure made of nodes, where each node has: data (the value) and pointer (link to next node).
    maintain dynamic list of active users/devices connected to a tower
    Space Efficiency: Efficient for insert/delete
    Time Efficiency: 0(n)

  • Hashing:
    Hashing maps input (like text, numbers, files) to a fixed-length of string.
    Customer ID, billing and IP mapping lookups. Space Efficiency:Extra space
    Time Efficiency: O(1) average

  • Graph:
    A graph in DAA is a non-linear data structure made of:
    Vertices (nodes) represent entities
    Edges (links) represent connections
    Network topology representation
    Space Efficiency: Moderate
    Time Efficiency: Optimal path computation.

  • Dijikstra:
    Dijkstra’s algorithm finds the shortest path from a source node to all other nodes in a weighted graph.
    Find shortest route between routers/fiber nodes
    Space Efficiency:Low
    Time Efficiency:O(E log V)

  • Prim’s:
    Prim’s algorithm builds a Minimum Spanning Tree (MST) by starting from one vertex and repeatedly adding the smallest edge that connects a new vertex to the growing tree, until all vertices are included.
    Build Minimum Spanning Tree for fiber layout
    Space Efficiency:Moderate
    Time Efficiency: O(E log V)

  • Kruskal’s
    Kruskal’s algorithm builds a Minimum Spanning Tree (MST) by sorting all edges by weight and then adding them one by one, as long as they don’t form a cycle.
    Cost-efficient tower placement & wiring route. Space Efficiency:Moderate
    Time Efficiency: O(E log V)

  • Quick Sort / Merge Sort:
    Sorting algorithms are fundamental algorithms used to arrange elements in a specific order (usually in ascending or descending order).
    Sort tower distances, bandwidth, user database.
    Space Efficiency:Low
    Time Efficiency: O( n log n)


Techniques used:

  • Divide & Conquer :
    Divide and Conquer is an algorithmic strategy that breaks a problem into smaller subproblems, solves them recursively, and then combines their solutions.
    Divide city into zones and optimize routing zone-wise.
  • Greedy Technique:
    The Greedy technique is an algorithmic approach that makes the locally optimal choice at each step.
    Choose lowest-cost next tower while laying fiber (Prim/Kruskal)

  • Backtracking:
    Backtracking is an algorithmic technique used to solve problems by exploring all possible solutions step by step and undoing (backtracking) choices when they lead to a dead end.
    When path fails, find alternative routing path.

Principles used:

  • Brave & Cautious Travel (DFS & BFS)
    DFS (Depth‑First Search) explores paths deeply, like a brave traveler venturing far before backtracking. BFS (Breadth‑First Search) explores level by level, cautiously covering all nearby nodes before moving outward.

  • Pruning:
    Pruning means cutting off unnecessary branches in a search tree. It avoids exploring paths that cannot lead to a valid or optimal solution.

  • Edge Relaxation: - Edge relaxation updates the shortest path estimate between two vertices. It’s the key step in algorithms like Dijkstra’s and Bellman‑Ford.

Agriculture

Agriculture is the practice of cultivating crops, raising animals, and managing natural resources to produce food and raw materials.

1. Drone Based Crop Monitoring and spraying: This idea aligns with the United nations Sustainable Development Goal (SDG) SDG 2 – Zero Hunger, SDG 12 – Responsible Consumption & Production, SDG 9 – Industry, Innovation & Infrastructure , SDG 15 – Life on Land

How it Solves Citizens and Local Business Problems:

  • Detects crop disease, nutrient deficiency, pest infection early via aerial survey
  • Saves time, labour & cost by automated pesticide and fertilizer spraying
  • Reduces chemical usage up to 50% using targeted spraying
  • Prevents crop loss → Higher yield & higher income
  • Helps government & farmers track agricultural productivity & weather impact
  • Enables smart farming & data-driven decisions


Algorithms/data structures used:
  • Structures:
    Store crop info: location, health status, moisture, spray level.
    Space Efficiency:Efficient
    Time Efficiency: Fast record handling

  • Heap (Priority Queue):
    A heap is a special tree-based structure where the parent node is ordered relative to its children. A priority queue uses a heap to efficiently access and remove the element with the highest (or lowest) priority.
    Prioritize affected crop zones to spray first.
    Space Efficiency:Moderate
    Time Efficiency: 0(log n)

  • Lookup table:
    A lookup table (LUT) is a precomputed list of values that a program can quickly “look up” instead of recalculating them every time.
    Quick lookup for spray levels and crop types.
    Space Efficiency:0(n)
    Time Efficiency: 0(1)

  • Hashing :
    Hashing is the process of converting data into a fixed‑size value called a hash using a hash function.
    Used for obstacle-free path exploration.
    Space Efficiency:O(V + E)
    Time Efficiency: 0(1) average

  • BFS :
    Breadth‑First Search is a graph traversal algorithm that explores level by level, starting from a source node.
    Store visited positions and avoid re-scanning.
    Space Efficiency:0(n)
    Time Efficiency: O(V + E)

  • Dijkstra :
    Dijkstra’s Algorithm is used to find the shortest path from a source node to all other nodes in a weighted graph (with non‑negative weights).
    For shortest path from base → field → recharge point.
    Space Efficiency:O(V + E)
    Time Efficiency: O(V + E)


Techniques used:
  • Divide and conquer :
    The divide‑and‑conquer technique is used to split the agricultural field into smaller sections to ensure efficient coverage.

  • Greedy :
    A greedy strategy selects the nearest grid cell at each step to plan the drone’s movement path.

  • Dynamic Programming:
    Dynamic programming helps predict crop health by storing and reusing solutions to repeated subproblems.

  • Space–Time Tradeof: A space–time tradeoff is applied by using structures like Grid HashMaps or Segment Trees to speed up field data queries.

Principles used:
  • Pruning:
    Pruning eliminates unnecessary or unproductive grid visits to optimize the drone’s traversal.

  • Hierarchical Data :
    Hierarchical data structures organize the farm into a Zone → Field → Grid tree for efficient navigation and analysis.

  • Edge Relaxation :
    Edge relaxation is used in Dijkstra’s algorithm to compute the shortest path for drone movement.

  • Memoization :
    Memoization saves repeated crop‑health calculations to avoid redundant processing.

  • Sliding Window:
    The sliding‑window technique monitors crop‑health time‑series data by analyzing continuous intervals efficiently.
